# The Grid Search (HackerRank)

## Problem Description

You are given two 2D grids of digits:  
- **G**, the larger grid.  
- **P**, the pattern grid.

You must determine whether **P** appears as a contiguous subgrid within **G**.

If it does, print **"YES"**.  
Otherwise, print **"NO"**.
